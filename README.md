## Project Overview

This repository contains my Digital Literacy Portfolio developed as part of the CSE0001 course. The project demonstrates essential digital skills including online safety, professional communication, coding basics, and cybercrime awareness.

As a Student Digital Ambassador, this portfolio reflects my ability to use digital tools responsibly and guide others in the digital environment.


## Module Summaries

### Module 1: Digital Literacy Awareness

This module introduced the concept of digital literacy and its importance in modern education. It focused on safe internet practices, responsible online behavior, and the use of digital tools for learning.

### Module 2: Digital Portfolio Development

This module emphasized building a professional online presence using platforms like GitHub and LinkedIn. It highlighted how digital profiles can support academic and career growth.

### Module 3: Coding & Collaboration Platforms

This module focused on developing basic coding skills and using collaboration tools. It included solving problems on platforms like HackerRank and creating Google Forms for data collection.

### Module 4: Email Etiquette & Communication

This module covered the principles of professional communication, including writing formal emails and maintaining proper tone and structure in digital interactions.

### Module 5: Cybercrime Awareness

This module focused on identifying common cyber threats such as UPI fraud and phishing. It also emphasized preventive measures and safe online practices.
